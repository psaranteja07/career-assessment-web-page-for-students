# Career Assessment Platform

A React (Vite) web application that enables students to take career assessments, view recommendations and courses, and provides an admin dashboard for managing tools and viewing results.

## Tech Stack
- React 18, Vite
- React Router
- Context API for state management
- MUI for styling

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Features
- Login page with role selection (student/admin)
- Option selection page (Take Assessment, Admin Dashboard)
- Assessment page with questions, result computation, and recommended courses section
- Admin dashboard to manage tools and view sample results
- Responsive design and accessible color contrast

## Notes
- This demo uses client-side state only (no backend). Integrate APIs for persistence.
